# Weather Report App - ReactJS

This is a simple ReactJS application that fetches weather data from the OPENWEATHER API.

## Important Notes:
  - Uses City(and Country) information to get weather data
  - dependecies: 
    "axios": "^0.18.0", <br>
    "react": "^16.8.6", <br>
    "react-dom": "^16.8.6", <br>
    "react-scripts": "2.1.8" <br>
